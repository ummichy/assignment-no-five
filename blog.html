Question-1: What are the different ways to select an element in the DOM?

Elements can be selected using getElementById() for IDs, getElementsByClassName() for classes, and getElementsByTagName() for tags. querySelector() and querySelectorAll() allow selection using CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML inside an element, including tags. innerText gets only the visible text, ignoring hidden content. textContent gets all text, even if hidden, but does not process HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation means adding an event listener to a parent instead of each child. When a child element triggers the event, it reaches the parent, which then handles it. This makes the code simpler and works well for dynamic elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves up to its parent elements. If a child element has an event, the same event also triggers on its parent and continues up the DOM tree unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

Elements can be created using createElement(). After creation, they can be added to the DOM using appendChild(). To remove an element, removeChild()can be used.These methods help in dynamically updating the webpage.
